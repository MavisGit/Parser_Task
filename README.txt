# Parser_Task
This work is a parser for the site himbio.ru. The parser takes data about products on the store's site (total ~ 1700 items at the time of script creation).
End result:
*data/img - directory with images of goods in .png format;
*data/csv/all_item - csv-file with information about all products;
*data/csv/items_card - directory where cards with information about each individual product are stored.
